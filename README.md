## Установка

- Скачать архив по ссылке выше (не разархивировать!)
- Открыть Blender
- Открыть меню `Edit -> Preferences...`
- Открыть секцию `Add-ons`
- Нажать на кнопку `Install...` сверху интерфейса
- Выбрать скачанный архив
